# OhMyGrill Brasas 🔥

E-commerce web para OhMyGrill Brasas — Zaragoza, España.

## Stack
- React 18 + Vite
- Stripe (pagos)
- Supabase (base de datos / pedidos)
- Google Maps API (zonas de reparto en Zaragoza)

## Desarrollo local

```bash
npm install
npm run dev
```

## Build producción

```bash
npm run build
npm run preview
```
